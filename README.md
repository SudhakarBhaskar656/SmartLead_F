SmartLead Frontend (React.js + vite)

This is the frontend UI for the SmartLead Automation System.
It allows users to:

Enter names for processing
Submit batch to backend
View live enriched leads
Filter by status
Styled using external CSS



Tech Stack

React
Axios
CSS3

1. Installation & Setup

Clone repo & navigate to frontend
git clone https://github.com/sudhakarbhaskar656/SmartLead_F.git
cd smartlead/frontend

2. Install dependencies

npm install

3. Start the React app

npm start
